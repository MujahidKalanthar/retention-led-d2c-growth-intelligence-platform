# Bla Bli Blu — Retention-Led D2C Growth Intelligence Platform

An end-to-end analytics and strategic intelligence platform designed to model, analyze, and optimize retention-led growth for premium D2C fragrance brands.

## Project Objectives

- Analyze customer lifecycle behavior
- Optimize trial-to-full-size conversion
- Model retention and LTV growth
- Simulate pricing and discount strategies
- Build executive-level business intelligence dashboards
- Generate strategic insights for sustainable premium growth

## Core Features

- Executive KPI dashboards
- Customer segmentation
- Cohort retention analysis
- Pricing simulations
- Campaign analytics
- Sentiment analysis
- Market intelligence dashboards
- Scenario modeling

## Tech Stack

- Python
- Pandas
- SQL
- Streamlit
- Plotly
- Scikit-learn
